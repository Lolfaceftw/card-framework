# notebooks/

This directory contains Jupyter notebooks for experimentation, testing, and demonstration of the audio-splitter functionality.

## Setup

1. Install Jupyter: `pip install jupyter`
2. Launch notebook server: `jupyter notebook` from the project root
3. Open notebooks from this directory

## Usage

Use notebooks to:
- Test audio separation on sample data
- Experiment with different parameters
- Visualize separation results
- Debug issues with the pipeline

## Dependencies

Ensure all requirements from `requirements.txt` are installed before running notebooks.
